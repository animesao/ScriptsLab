# ⚔️ МЕЧ МОЛНИЙ - Руководство

## 🎯 Описание

**Меч Молний** - легендарный кастомный предмет с уникальными способностями, атрибутами и эффектами.

## 📋 Как получить

Используйте команду:
```
/getlightningsword
```

**Требуемое разрешение:** `scriptslab.getlightningsword`

## ⚡ Способности

### 1. **Молния при ударе**
- При ударе по мобу/игроку вызывается молния в точке цели
- Наносит дополнительный урон: **+5 ❤**
- Эффектно и смертельно!

### 2. **Эффект Силы**
- При взятии меча в руку игрок получает эффект **Сила II**
- Эффект действует бесконечно, пока меч в руке
- При убирании меча эффект исчезает
- Частицы эффекта скрыты для чистоты

## 📊 Характеристики

### Атрибуты (Attributes)
- **Урон:** +10 ❤ (дополнительно к базовому урону алмазного меча)
- **Скорость атаки:** +20% (+0.8 к базовой скорости)
- **Прочность:** Неразрушимый (Unbreakable)

### Визуальные эффекты
- ✨ **Свечение** (Enchant Glow) - меч светится как зачарованный
- 🎨 **Кастомное название:** §6§l⚡ МЕЧ МОЛНИЙ ⚡
- 📜 **Детальное описание** (Lore) с информацией о способностях

### Скрытые флаги
- Скрыты атрибуты (чтобы не загромождать описание)
- Скрыта неразрушимость
- Скрыты зачарования

## 🎮 Использование

### Получение меча
1. Введите `/getlightningsword`
2. Меч появится в вашем инвентаре
3. Возьмите меч в руку

### Активация способностей
1. **Молния:** Просто ударьте любого моба или игрока
2. **Сила:** Автоматически активируется при взятии меча в руку

### Советы
- Используйте меч против сильных мобов (Wither, Ender Dragon)
- Молния наносит урон по площади - можно убить несколько мобов сразу
- Эффект Силы II значительно увеличивает урон в ближнем бою
- Меч неразрушимый - используйте без ограничений!

## 🔧 Технические детали

### Реализация
- **Язык:** JavaScript (GraalVM)
- **API:** Bukkit/Paper 1.21.8
- **События:** EntityDamageByEntityEvent, PlayerItemHeldEvent

### Атрибуты (AttributeModifier)
```javascript
// Урон
Attribute.GENERIC_ATTACK_DAMAGE: +10.0 (ADD_NUMBER)

// Скорость атаки
Attribute.GENERIC_ATTACK_SPEED: +0.8 (ADD_NUMBER)
```

### Эффекты (PotionEffect)
```javascript
PotionEffectType.STRENGTH
Уровень: 1 (= Сила II)
Длительность: 999999 тиков (~13.8 часов)
Ambient: false
Particles: false (скрыты)
```

## 🛠️ Кастомизация

Вы можете изменить меч в файле `scripts/custom_sword.js`:

### Изменить урон
```javascript
var damageModifier = new AttributeModifier(
    UUID.randomUUID(),
    'lightning_sword_damage',
    15.0, // Измените это значение
    AttributeModifier.Operation.ADD_NUMBER,
    EquipmentSlot.HAND
);
```

### Изменить эффект
```javascript
player.addPotionEffect(new PotionEffect(
    PotionEffectType.SPEED, // Измените тип эффекта
    999999,
    2, // Измените уровень (0 = I, 1 = II, 2 = III)
    false,
    false
));
```

### Добавить другие способности
Вы можете добавить:
- Огненный урон (setFireTicks)
- Телепортацию (player.teleport)
- Взрыв (world.createExplosion)
- Лечение (player.setHealth)
- И многое другое!

## ⚠️ Известные особенности

1. **Молния в креативе:** Молния вызывается даже в креативном режиме
2. **Эффект Силы:** Не стакается с другими эффектами Силы
3. **Неразрушимость:** Меч нельзя сломать даже в /kill

## 🎯 Баланс

Меч очень мощный! Для баланса можно:
- Добавить кулдаун на молнию
- Уменьшить урон
- Добавить стоимость использования (опыт, предметы)
- Ограничить количество использований

## 🔮 Идеи для улучшения

1. **Кулдаун:** Молния раз в 5 секунд
2. **Мана:** Использование тратит опыт
3. **Прокачка:** Меч улучшается с использованием
4. **Комбо:** Серия ударов = более мощная молния
5. **Режимы:** ПКМ переключает режимы (молния/огонь/лёд)
